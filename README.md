# skillpod

A clean and modern startup landing page built using HTML, CSS, and JavaScript.
Skill Pod is a simple idea focused on helping people learn new skills in an easy, structured, and engaging way.

##  About the Project

Skill Pod is designed as a startup landing page that presents a learning platform idea.  
The goal is to make skill learning simple and accessible for everyone.

This project focuses on:
- Clean UI design
- Simple and clear content
- Smooth user experience

## Problem

Many people want to learn new skills but face problems like:
- Too much unorganized content online  
- Lack of proper guidance  
- Difficulty in tracking progress  

##  Solution

Skill Pod solves this by providing:
- Structured learning approach  
- Simple and easy-to-use interface  
- Clear learning path for users  

##  Features

- Modern and responsive landing page  
- Smooth scrolling navigation  
- Simple animations (fade-in effects)  
- Clean layout with proper sections  
- Contact form UI  

##  Tech Used

- HTML  
- CSS  
- JavaScript
